# BrickBreaker

A BreakBreaker game in java, made following Object Oriented Programming best practices and Design Patterns

## Quickstart Command

```
java -XX:+ShowCodeDetailsInExceptionMessages -cp $dir$/DrickBreaker/bin Main.Application
```
